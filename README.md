# Aqa-PlaylistApp

## Get project to your computer

```
git clone https://github.com/Lekuro/Aqa-PlaylistApp.git
```

## Install

Open project in VSCode and type in terminal

```
npm install
```

### Create .env file according to .env.example

```
BASE_URL=https://vite-react-alpha-lemon.vercel.app/
```

## Run tests

```
npm run test
```
